Система обучения

  • ФОРМЫ ОБУЧЕНИЯ

    Самоподготовка по персональному учебному плану, дистанционные уроки, конференции, семинары, деловые игры, лабораторные работы, практикумы и другие формы учебных занятий, проводимых на базе платформы IBLS


  •      ТЬЮТОРЫ (НАСТАВНИКИ)

    Учащимся дистанционной формы могут быть предоставлены наставники (тьюторы -консультанты по образовательному процессу, которые оказывают ученикам помощь в организации учебной деятельности). Преподавателями в режиме он- и оффлайн осуществляют методическую поддержку по каждому предмету учебного плана. Это позволяет учащимся IBLS-Наши пенаты добиться высоких показателей при сдаче ОГЭ, ЕГЭ


  • УСТАНОВОЧНЫЕ КОНСУЛЬТАЦИИ

    При поступлении на программу дистанционного обучения учащийся получает первые установочные консультации, в ходе которых определюятся образовательные приоритеты и комфортный режим обучения ребенка


Образовательный процесс и аттестация

Обучение и тестирование

Учащиеся дистанционной формы, проходят темы учебного плана в режиме самоподготовки. По итогам самоподготовки учащиеся проходят закрепляющие тематические тесты. Оценки выставляются в электронный журнал

АТТЕСТАЦИЯ

Аттестация проводится 2 раза в год для 1, 10, 11 классов (по полугодиям) и 3 раза в год (по триместрам) для 2-9 классов

КОНТРОЛЬНЫЕ РАБОТЫ

Для прохождения аттестации учащийся дистанционно выполняет контрольные работы через систему IBLS. В случае наличия проблем с интернет-соединением, задания направляются по электронной почте, учащийся направляет в ответ сканированные листы с выполненной работой, либо организует доставку оригиналов на проверку в школу
